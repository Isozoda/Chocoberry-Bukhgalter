# Choco Berry Business Management System

Ин лоиҳа як системаи мукаммали идоракунии тиҷорат барои "Choco Berry" мебошад. Он аз ду қисми асосӣ иборат аст: **Backend** (API) ва **Frontend** (Интерфейси корбар).

## Сохтори лоиҳа
Лоиҳа ба ду қисм тақсим шудааст:
- `backend/`: Сервер ва махзани маълумот (NestJS + Prisma).
- `chocoberry-frontend/`: Интерфейси корбар (React + Vite).

## Мақсади лоиҳа
Автоматикунонии равандҳои тиҷоратии Choco Berry, аз ҷумла фурӯш, идоракунии анбор, ҳисобкунии маош ва ҳисоботҳои молиявӣ.
