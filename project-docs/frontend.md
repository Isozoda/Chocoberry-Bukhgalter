# Маълумот дар бораи Frontend (MoneyMind)

Frontend-и лоиҳаи MoneyMind (Choco Berry) як панели идоракунии муосир ва пурқувват аст, ки барои идоракунии тиҷорати шириниҳо сохта шудааст. Он бо истифода аз технологияҳои навтарин ва беҳтарин таҷрибаҳои UI/UX таҳия шудааст.

## 🛠 Технологияҳо (Tech Stack)

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) (барои суръати фавқулодда ва таҷрибаи таҳияи беҳтар).
- **Language**: TypeScript (барои таъмини сифати код ва пешгирии хатогиҳо).
- **Styling**: 
  - [Tailwind CSS](https://tailwindcss.com/) (барои тарҳрезии зуд ва фасеҳ).
  - [Shadcn UI](https://ui.shadcn.com/) (компонентҳои муосир дар асоси Radix UI).
  - [Material UI (MUI)](https://mui.com/) (барои иконкаҳо ва баъзе компонентҳои махсус).
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) (идоракунии ҳолати глобалӣ, содда ва тез).
- **Data Fetching**: [TanStack Query (React Query)](https://tanstack.com/query/latest) ва Axios (барои иртибот бо API ва кэшкунии маълумот).
- **Form Handling**: React Hook Form ва Zod (барои вализатсияи формаҳо).
- **Internationalization (i18n)**: i18next (дастгирии пурраи забонҳои **Тоҷикӣ, Русӣ ва Англисӣ**).
- **Charts**: Recharts (барои графикҳои молиявӣ ва таҳлилӣ).

## 📂 Сохтори Лоиҳа

- `src/api/`: Функсияҳо барои иртибот бо Backend.
- `src/components/`: Компонентҳои такроршаванда (UI, Layout, Forms, Charts).
- `src/hooks/`: Custom hooks барои мантиқи такроршаванда.
- `src/pages/`: Саҳифаҳои асосии барнома (модулҳо).
- `src/store/`: Store-ҳои Zustand барои идоракунии ҳолат (Auth, Theme, UI).
- `src/i18n/`: Файлҳои тарҷума ва танзимоти забон.

## 🚀 Модулҳои Асосӣ (Modules)

1. **Dashboard (Панели асосӣ)**:
   - Нишондиҳандаҳои калидии фурӯш (Sales KPIs).
   - Огоҳиҳо дар бораи камшавии захираҳои анбор (Low Stock).
   - Ҳолати хазина (Cashbox Balance).
   - Графикҳои фурӯши ҳафтаина.

2. **Sales (Фурӯш/POS)**:
   - Терминали фурӯш (POS) бо имконияти иловаи маҳсулот ба сабад.
   - Дастгирии намудҳои пардохт: Нақдӣ, Кортӣ ва Омехта (Mixed).
   - Таърихи фурӯшҳо ва имконияти бекоркунӣ (Void).

3. **Inventory (Анбор)**:
   - Идоракунии ашёи хом (Меваҳо, Шоколад ва ғайра).
   - Раванди тозакунии маҳсулот (Cleaning) бо ҳисоби талафот (Loss weight).
   - Баҳисобгирии масрафҳо ва талафот (Waste).

4. **Products (Маҳсулот)**:
   - Рӯйхати маҳсулот (десертҳо) аз рӯи ҳаҷми истаконҳо.
   - Созандаи рецепт (Recipe Builder/BOM) барои ҳисоби худкори арзиши аслӣ (Cost).

5. **Finance (Молия)**:
   - **Expenses**: Масрафҳои доимӣ ва тағйирёбанда.
   - **Cashbox**: Идоракунии хазинаи нақдӣ ва бонкӣ.
   - **Funds**: Фондҳои махсус (Хайрия, Захира, Ободонӣ ва ғайра).

6. **Staff (Кормандон)**:
   - Идоракунии кормандон ва нақшҳои онҳо.
   - Давомот (Attendance) ва ҷадвали корӣ.
   - Ҳисоби музди меҳнат (Payroll) бо назардошти мукофотпулӣ (Bonus) ва ҷаримаҳо.

7. **Reports (Ҳисоботҳо)**:
   - Ҳисоботи фоида ва зарар (P&L).
   - Таҳлили фурӯшҳо аз рӯи соат ва маҳсулот.
   - Имконияти гирифтани ҳисобот дар шакли **Excel** ва **PDF**.

8. **AI Advisor (Машваратчии AI)**:
   - Интегратсия бо AI барои таҳлили додаҳо ва додани тавсияҳо барои тиҷорат.

9. **Telegram Integration**:
   - Пайваст кардани профил бо боти Telegram барои гирифтани огоҳиҳо.

## ✨ Хусусиятҳои Дигар

- **Dark Mode**: Интерфейси торик ва равшан.
- **Responsive Design**: Барнома дар телефон, планшет ва компютер хуб кор мекунад.
- **Real-time Validation**: Санҷиши маълумот ҳангоми ворид кардан.
