# Маълумот дар бораи Frontend

Frontend-и лоиҳа барои таҷрибаи беҳтарини корбар бо технологияҳои тез сохта шудааст.

## Технологияҳо
- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) (барои суръати баланди кор).
- **Language**: TypeScript (барои амнияти код).
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) ва [Material UI](https://mui.com/).
- **State Management**: [Zustand](https://github.com/pmndrs/zustand).
- **Data Fetching**: [React Query](https://tanstack.com/query/latest) ва Axios.
- **Form Handling**: React Hook Form ва Zod.
- **Internationalization (i18n)**: i18next (барои дастгирии забонҳои Тоҷикӣ, Русӣ ва Англисӣ).
- **Charts**: Recharts (барои нишон додани графикҳои молиявӣ).

## Интерфейс
- Дизайни муосир бо истифода аз компонентҳои Radix UI.
- Интерфейси торик (Dark Mode).
- Дизайни респонсив (барои телефон ва планшет мувофиқ).
