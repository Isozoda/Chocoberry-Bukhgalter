# Маълумот дар бораи лоиҳаи Choco Berry

Ман ҳамаи файлҳои лоиҳаро таҳлил кардам ва дар папкаи `project-docs` маълумоти муфассалро ҷойгир намудам. Дар инҷо хулосаи кӯтоҳ оварда шудааст:

## 1. Сохтори умумии лоиҳа
Лоиҳа аз ду қисми асосӣ иборат аст:
- **Backend**: Дар папкаи `backend/` ҷойгир аст.
- **Frontend**: Дар папкаи `chocoberry-frontend/` ҷойгир аст.

## 2. Backend (Сервер ва Махзани маълумот)
- **Технология**: NestJS (Node.js framework).
- **Махзани маълумот**: Prisma ORM (барои кор бо PostgreSQL).
- **Амният**: JWT ва Passport.js барои воридшавии корбарон.
- **Функсияҳо**: API барои фурӯш, анбор, маош ва ҳисоботҳо.

## 3. Frontend (Интерфейси корбар)
- **Технология**: React 19 + Vite.
- **Дизайн**: Tailwind CSS ва Material UI (MUI).
- **Забонҳо**: Дастгирии забонҳои Тоҷикӣ, Русӣ ва Англисӣ (i18next).
- **Ҳолат (State)**: Zustand ва React Query.

## 4. Имкониятҳои асосӣ
- **POS**: Барои фурӯши маҳсулот.
- **Inventory**: Назорати анбор.
- **Payroll**: Ҳисобкунии маоши кормандон.
- **P&L Analytics**: Таҳлили фоида ва зарар.
