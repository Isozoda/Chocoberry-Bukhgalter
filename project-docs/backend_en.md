# Backend Information

The project's backend is built using modern Node.js technologies.

## Technology Stack
- **Framework**: [NestJS](https://nestjs.com/) (for building robust and scalable APIs).
- **ORM**: [Prisma](https://www.prisma.io/) (for database interaction).
- **Database**: PostgreSQL (or any SQL database compatible with Prisma).
- **Authentication**: JWT (JSON Web Tokens) and Passport.js.
- **Validation**: class-validator and class-transformer.
- **Documentation**: Swagger (available at `/api/docs`).

## Core Functionality
1. **User Management**: Registration and Login (Auth).
2. **Financial Module**: P&L calculations, cashbox management.
3. **Inventory Management**: Tracking products and raw materials.
4. **Payroll & Employees**: Payroll module for calculating employee salaries.
5. **Reporting**: Generation of Excel and PDF reports.
