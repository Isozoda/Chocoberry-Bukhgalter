# Choco Berry Business Management System

This project is a comprehensive business management system for "Choco Berry". It consists of two main parts: **Backend** (API) and **Frontend** (User Interface).

## Project Structure
The project is divided into two parts:
- `backend/`: Server and database (NestJS + Prisma).
- `chocoberry-frontend/`: User interface (React + Vite).

## Project Goal
To automate Choco Berry's business processes, including sales, inventory management, payroll calculation, and financial reporting.
