# Frontend Information

The project's frontend is built for a premium user experience using fast and modern technologies.

## Technology Stack
- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) (for high performance).
- **Language**: TypeScript (for type safety).
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) and [Material UI](https://mui.com/).
- **State Management**: [Zustand](https://github.com/pmndrs/zustand).
- **Data Fetching**: [React Query](https://tanstack.com/query/latest) and Axios.
- **Form Handling**: React Hook Form and Zod.
- **Internationalization (i18n)**: i18next (supporting Tajik, Russian, and English).
- **Charts**: Recharts (for financial data visualization).

## Interface Features
- Modern UI using Radix UI components.
- Dark mode interface.
- Responsive design (mobile and tablet friendly).
