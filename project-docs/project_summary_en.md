# Choco Berry Project Information

I have analyzed all project files and placed detailed documentation in the `project-docs` folder. Here is a brief summary:

## 1. Overall Project Structure
The project consists of two main parts:
- **Backend**: Located in the `backend/` folder.
- **Frontend**: Located in the `chocoberry-frontend/` folder.

## 2. Backend (Server & Database)
- **Technology**: NestJS (Node.js framework).
- **Database**: Prisma ORM (working with PostgreSQL).
- **Security**: JWT and Passport.js for user authentication.
- **Functions**: API for sales, inventory, payroll, and reports.

## 3. Frontend (User Interface)
- **Technology**: React 19 + Vite.
- **Design**: Tailwind CSS and Material UI (MUI).
- **Languages**: Support for Tajik, Russian, and English (i18next).
- **State**: Zustand and React Query.

## 4. Key Features
- **POS**: For selling products.
- **Inventory**: Stock control.
- **Payroll**: Calculating employee salaries.
- **P&L Analytics**: Profit and Loss analysis.
