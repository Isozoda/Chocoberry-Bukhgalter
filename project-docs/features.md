# Имкониятҳои лоиҳа (Features)

Система дорои модулҳои зерин мебошад:

## 1. Dashboard (Панели асосӣ)
Нишондиҳандаҳои асосии тиҷорат дар шакли графикҳо ва рақамҳо.

## 2. POS (Нуқтаи фурӯш)
Интерфейс барои фурӯши маҳсулот ба мизоҷон.

## 3. Анбор (Inventory)
Назорати боқимондаи маҳсулот ва ашёи хом. Огоҳӣ дар мавриди кам мондани маҳсулот.

## 4. Рецептҳо (BOM)
Ҳисобкунии арзиши аслии маҳсулот дар асоси ашёи хоми истифодашуда.

## 5. Кормандон ва Маош (HR & Payroll)
Баҳисобгирии давомоти кормандон ва ҳисоб кардани маош дар асоси фурӯш ва ҷаримаҳо.

## 6. Хароҷот ва Даромад (Accounting)
Баҳисобгирии ҳамаи хароҷотҳои тиҷорат ва ҳисоб кардани фоидаи холис (P&L).

## 7. Ҳисоботи ҳаррӯза (Daily Reports)
Хулосаи кори якрӯзаи тиҷорат.
